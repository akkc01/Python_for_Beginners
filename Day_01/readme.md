### 🔹 2. Variables & Arithmetic Operations

```python
a = 50
b = 20

print(a + b)  # Addition
print(a - b)  # Subtraction
print(a * b)  # Multiplication
print(a / b)  # Division
print(a // b) # Floor Division
print(a % b)  # Modulus
print(a ** b) # Exponentiation
```

### 🔹 3. Comparison Operators

```python
print(a > b)
print(a < b)
print(a == b)
print(a != b)
print(a >= b)
print(a <= b)
```

---

### 🔹 4. Conditional Statements (if-elif-else)

#### Example 1:

```python
c = 100
if c > b and c > a:
    print("c is greater than a and b")
elif c > b or c > a:
    print("c is greater than either a or b")
else:
    print("c is not greater than a and b")
```

---

#### Example 2: Even/Odd Check

```python
z = 99
if z % 2 == 0:
    print("z is even")
else:
    print("z is odd")
```

---

#### Example 3: Age Classification

```python
age = 103
if age >= 18 and age <= 45:
    print("You are a Young Man")
elif age >= 60 and age <= 100:
    print("You are a Senior Citizen")
elif age <= 17:
    print("You are a child or a teenager")
else:
    print("You are Immortal")
```

---

#### Example 4: Grade System

```python
marks = 99

if marks >= 90:
    print("Grade A+")
elif marks >= 85:
    print("Grade A")
elif marks >= 75:
    print("Grade B+")
elif marks >= 65:
    print("Grade B")
elif marks >= 55:
    print("Grade C+")
elif marks >= 35:
    print("Grade C")
else:
    print("Fail")
```
